# Manual Security Lab

This repository documents a continuously evolving personal cybersecurity lab focused on hands-on attack, investigation and detection techniques.

## Lab Focus

- Manual log analysis  
- Network observation  
- Vulnerability discovery  
- Windows & Linux system investigation  
- Web application security testing
  
## Methodology

Each exercise follows a structured approach:

1. Identify attack surface  
2. Attempt exploitation  
3. Analyze system behavior  
4. Review logs and artifacts  
5. Document findings  

This lab is actively expanded as new scenarios and techniques are explored.
